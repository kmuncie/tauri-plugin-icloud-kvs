## Default Permission

Default permissions for the icloud-kvs plugin: allows every command.

#### This default permission set includes the following:

- `allow-get`
- `allow-set`
- `allow-remove`
- `allow-keys`
- `allow-get-all`
- `allow-synchronize`
- `allow-account-status`
